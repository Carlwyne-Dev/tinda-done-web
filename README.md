# TindaDone Web

The official landing page for **TindaDone** - an offline-first inventory and POS application. 

## Features
- **Modern UI**: Built with React, Vite, and Framer Motion for buttery-smooth animations.
- **Responsive**: Fully optimized for mobile and desktop screens.
- **Parallax & Marquee**: Features a scroll-driven infinite parallax mockup viewer.
- **Direct APK Download**: Easy access to the official TindaDone APK.

## Tech Stack
- React 18
- Vite
- Framer Motion
- Lucide React Icons

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run the Development Server**
   ```bash
   npm run dev
   ```

3. **Deploy**
   Build the project for production:
   ```bash
   npm run build
   ```

## Adding the APK
Make sure to place the `tindadone.apk` file inside the `public/` folder so it can be downloaded directly from the main download button.
